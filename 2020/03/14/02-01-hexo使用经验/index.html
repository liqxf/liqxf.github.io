添加新文章

  1. hexo new “new_file_name”
    会在source/_posts/目录下创建一个新文件： new_file_name.md
  2. hexo g // generate 新页面
  3. hexo s // 在localhost运行serer， 此时可以在浏览器上查看： http://localhost:4000/
  4. hexo d // 部署到github

hexo本地和github不一致

  1. hexo clean
  2. hexo d -g
